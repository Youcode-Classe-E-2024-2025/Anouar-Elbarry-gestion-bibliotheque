# 📚 Library Management System

## 🌟 Project Overview

A modern, efficient web application for library management built using Laravel, designed to simplify book borrowing, user authentication, and library operations.

## ✨ Key Features

### User Features
- 🔐 Secure User Authentication
- 👤 Profile Management
- 📖 Book Browsing
- 📚 Book Borrowing System
- 📋 Personal Borrowed Books Tracking
- 📱 Fully Responsive Design

### Admin Features
- 🖥️ Admin Dashboard
- 📊 Book Inventory Management
- 👥 User Management

## 🛠 Technical Stack

- **Backend**: Laravel 10.x
- **Frontend**: Blade Templates, Tailwind CSS
- **Database**: PostgreSQL
- **Authentication**: Laravel's Built-in Auth System
- **Design**: Mobile-first Responsive Layout

## 🚀 Quick Start

### Prerequisites
- PHP 8.1+
- Composer
- PostgreSQL
- Node.js & npm

### Installation Steps

#### Install Dependencies
```bash
composer install
npm install
```

#### Environment Setup
```bash
cp .env.example .env
php artisan key:generate
```

#### Database Configuration
```bash
php artisan migrate
php artisan db:seed
```

#### Run the Application
```bash
php artisan serve
npm run dev
```
